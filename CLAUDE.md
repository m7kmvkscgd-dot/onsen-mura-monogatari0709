# 温泉村物語

`C:\ダンジョン1\` を複製して作った新プロジェクト。ゲームの仕組み(パーティ制・パーマデス・戦闘システム等)はダンジョン1から引き継ぎつつ、**和製(和風)の方向性に作り変えていく**ことが目的。ダンジョン1自体は元のまま残っており、このプロジェクトはそこから分岐した独立した作業ブランチ。

## ダンジョン1から引き継いだ仕組み(ゲームエンジン部分は共通)
- 4人パーティ制、ターン制バトル
- MP制(魔法系だけでなく物理必殺技もMP消費。町でのみ回復)
- 疲労度システム(潜り続けると弱体化、町で回復)
- パーマデス+死体回収システム(死んだ場所に死体が残り、時間制限内に聖水で救出しないとロスト)
- 複数敵・範囲攻撃(1〜3体の集団が出現し、範囲技持ちが雑魚の群れに有効)
- 職業ごとの武器/防具(3段階、レベルで解禁、購入すると同職業全員に永久ステータス加算)
- ダンジョン探索は「進む」ボタンのみの線形進行(方向選択・グリッド移動なし)
- スマホ前提のレイアウト

## 和製化で変更した内容(職業を西洋ファンタジー8種→和風8種に総入れ替え済み)
ユーザー提供のキャラシート(`assets/class_sheet_source_jp.png`)に基づき、職業を以下の8種に総入れ替えした:
- **侍**(会心の一撃・単体特化) / **忍**(奇襲・高速単体) / **槍士**(かばう・タンク、旧パラディン相当の防御役)
- **薙刀士**(薙ぎ払い・範囲物理) / **狩人**(会心の一矢・単体、防御貫通の弓技)
- **砲術士**(砲撃・超火力だが使うと次のターンは装填で動けない、新規メカニクス) / **陰陽師**(呪符ノ術/大祓ノ術、旧魔法使い相当) / **僧侶**(治癒の術、変更なし)

職業ID(`data.js`の`CLASSES`キー)も `samurai/ninja/spearman/naginata/hunter/gunner/onmyoji/priest` に変更済み。`EQUIPMENT`カタログも同じ8IDで作り直し済み(和風武具名)。キャラ絵は`assets/class_samurai.png`等(8種)。
